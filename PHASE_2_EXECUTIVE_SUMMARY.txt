HOW TO USE (Extremely simple!)
════════════════════════════════════════════════════════════════════════════════

TO ADD A NEW PROJECT:
  1. Open: src/data/projects.json
  2. Add a new object to the "projects" array
  3. Save
  4. Done! (Appears immediately on site)

TO ADD A NEW BLOG POST:
  1. Create: src/content/blog/my-post.md
  2. Add YAML front-matter (title, date, tags, excerpt)
  3. Write markdown content
  4. Save
  5. Done! (Appears immediately in blog section)

IN YOUR REACT APP:
  import { ProjectsSection, BlogSection } from '@/components/sections';
  
  function HomePage() {
    return (
      <>
        <ProjectsSection />  {/* That's it! */}
        <BlogSection />      {/* All content loads automatically */}
      </>
    );
  }
