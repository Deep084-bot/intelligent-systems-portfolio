# Guitar Audio Samples

This directory should contain 6 high-quality acoustic guitar samples for standard tuning.

## Required Files

- `e2.mp3` - Low E string (6th string)
- `a2.mp3` - A string (5th string)
- `d3.mp3` - D string (4th string)
- `g3.mp3` - G string (3rd string)
- `b3.mp3` - B string (2nd string)
- `e4.mp3` - High E string (1st string)

## Sample Requirements

- Format: MP3 (widely supported, small file size)
- Length: 2-4 seconds (natural decay)
- Quality: 128kbps minimum, 192kbps recommended
- Recording: Clean acoustic guitar, no effects
- Trim: Remove silence at start, keep natural decay at end

## Sources

### Free Options
1. **Freesound.org** - Search for "acoustic guitar single note"
2. **Musical Artifacts** - Public domain instrument samples
3. **Record yourself** - Use phone/mic and a real guitar

### Paid Options
1. **Splice Sounds** - Professional guitar sample packs
2. **Loopcloud** - Acoustic guitar one-shots
3. **Native Instruments** - Kontakt libraries
